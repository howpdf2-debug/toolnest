GST Invoice Kya Hoti Hai?

GST invoice ek legal document hoti hai jo ek registered GST business apne customer ko deta hai jab woh koi product bechta hai ya service provide karta hai. Yeh document prove karta hai ki transaction hua hai aur kitna GST charge kiya gaya hai.

India mein GST (Goods and Services Tax) July 2017 mein launch hua tha. Tab se har registered business ke liye yeh zaroori hai ki woh apni har sale par GST invoice issue kare. Agar aap invoice issue nahi karte toh GST department se notice aa sakti hai aur penalty bhi lag sakti hai.

GST Invoice Kiske Liye Zaroori Hai?

  • GST registered businesses: Annual turnover Rs 40 lakh (goods) ya Rs 20 lakh (services) se zyada hone par
  • E-commerce sellers: Amazon, Flipkart, Meesho par sell karne wale sabhi sellers
  • Freelancers: Agar annual income Rs 20 lakh se zyada hai
  • Exporters: Export business ke liye bhi GST invoice compulsory hai

CGST, SGST aur IGST Mein Kya Fark Hai?

Yeh sabse common confusion hai. Simple explanation:

Same State Transaction (Intra-state):

Jab aap aur aapka customer same state mein hain — CGST + SGST lagta hai:

  • 18% GST wali item: CGST 9% + SGST 9%
  • 12% GST wali item: CGST 6% + SGST 6%
  • 5% GST wali item: CGST 2.5% + SGST 2.5%

Different State Transaction (Inter-state):

Jab aap aur customer alag states mein hain — sirf IGST lagta hai (Mumbai seller → Delhi customer = IGST).

💡
Easy Rule: Same state customer ko becho → CGST + SGST. Doosre state ke customer ko becho → IGST. Export karo → 0% tax (IGST with zero rate).

HSN Code Kahan Se Milega?

  • GST Portal: gst.gov.in → Services → Search HSN Code
  • Google search: "HSN code for [product name] India"

Kitne Digits Ka Code Chahiye?

  • Annual turnover Rs 5 crore tak: 4-digit HSN code sufficient
  • Rs 50 crore se zyada: 6-digit HSN code compulsory

Common GST Invoice Mistakes — Inse Bachein

  • ❌ Galat GSTIN: Customer ka GSTIN carefully verify karein — galat number se unhe ITC nahi milega
  • ❌ Invoice sequence todna: Invoice numbers sequential hone chahiye — gaps avoid karein
  • ❌ Galat GST rate: Apne product/service ka correct rate confirm karein
  • ❌ HSN code missing: Turnover ke hisaab se HSN code compulsory hai
  • ❌ Due date miss karna: GST returns time par file karein — late filing par interest aur penalty lagti hai
💡
Record keeping tip: GST law ke according sabhi invoices minimum 6 years tak rakhne chahiye. Digital copies organized folders mein save karo — month-wise ya year-wise.
