GST Invoice Kya Hoti Hai?

GST invoice ek legal document hoti hai jo ek registered GST business apne customer ko deta hai jab woh koi product bechta hai ya service provide karta hai. Yeh document prove karta hai ki transaction hua hai aur kitna GST charge kiya gaya hai.

India mein GST July 2017 mein launch hua. Tab se har registered business ke liye apni har sale par GST invoice issue karna legally compulsory hai. Invoice issue nahi kiya toh GST department se notice aur penalty dono aa sakti hain.

GST Invoice Kiske Liye Zaroori Hai?

  • GST registered businesses: Annual turnover Rs 40 lakh (goods) ya Rs 20 lakh (services) se zyada
  • E-commerce sellers: Amazon, Flipkart, Meesho par sell karne wale sabhi
  • Freelancers: Annual income Rs 20 lakh se zyada ho toh
  • Exporters: Export ke liye bhi GST invoice compulsory

CGST, SGST aur IGST — Simple Explanation

Yeh sabse common confusion hai. Ek baar samajh lo, hamesha ke liye clear ho jayega:

Same State Transaction (Intra-state)

Jab seller aur buyer same state mein hain — CGST + SGST dono lagte hain:

  • 18% GST item → CGST 9% + SGST 9%
  • 12% GST item → CGST 6% + SGST 6%
  • 5% GST item → CGST 2.5% + SGST 2.5%

Different State Transaction (Inter-state)

Jab seller aur buyer alag states mein hain — sirf IGST lagta hai. Mumbai seller → Delhi buyer = IGST only.

💡
Easy Rule: Same state = CGST + SGST. Doosra state = IGST only. Export = 0% (IGST with zero rate).

HSN Code Kahan Se Milega?

  • GST Portal: gst.gov.in → Services → Search HSN Code
  • Google: "HSN code for [product name] India" search karein
  • CA ya Tax Consultant: Agar naya business hai toh confirm karwa lein

Kitne Digits Ka HSN Code Chahiye?

  • Annual turnover Rs 5 crore tak → 4-digit HSN sufficient
  • Rs 5 crore se Rs 50 crore → 6-digit compulsory
  • Rs 50 crore se zyada → 8-digit compulsory

Common Mistakes — Inse Bachein

  • ❌ Galat GSTIN: Customer ka GSTIN verify karein — galat number se unhe ITC nahi milega
  • ❌ Invoice sequence todna: Numbers sequential hone chahiye — gaps nahi
  • ❌ Galat GST rate: Apne product/service ka correct rate confirm karein
  • ❌ HSN code missing: Turnover ke hisaab se compulsory hai
  • ❌ Due date miss: GST returns time par file karein — late filing par penalty lagti hai
💡
Record Keeping: GST law ke according sabhi invoices minimum 6 years tak rakhna zaroori hai. Digital copies month-wise ya year-wise save karo.
