1. Files Aur Documents

ToolNest ke sabhi tools 100% browser-based hain. Aapki koi bhi file — PDF, image, document — hamare servers par kabhi upload nahi hoti.

Processing sirf aapke browser (device) ki memory mein hoti hai. Page band karne par ya reload karne par files automatically delete ho jaati hain. Hum technically bhi aapki files dekh nahi sakte.

  • ✅ PDF merge, compress, convert — browser memory mein
  • ✅ Image compress, resize — Canvas API (local)
  • ✅ GST invoice data — sirf aapke device par
  • ✅ Calculator inputs — pure JavaScript, no server

3. Cookies Aur Local Storage

Hum use karte hain:

  • localStorage (browser-only): Dark mode preference aur UI state — ye sirf aapke browser mein rehta hai, server par nahi jaata.
  • Google Analytics cookies: Anonymized usage tracking. Aap browser settings mein in cookies ko block kar sakte hain.
  • Google AdSense cookies: Ads personalization ke liye (see section 4). Aap Google Ad Settings mein opt-out kar sakte hain.

Koi third-party tracking cookies nahi hain. Koi cross-site tracking nahi.

4. Advertisements

ToolNest ko free rakhne ke liye hum Google AdSense ads dikhate hain. Ye ads Google ki apni privacy policy ke according serve hoti hain.

Important points:

  • Hum advertiser data ko aapke tool usage se link nahi karte
  • Aap ads personalization disable kar sakte hain
  • Ad revenue sirf server costs aur development mein jaata hai

More: Google Privacy Policy →

5. Data Security

Kyunki hum aapka sensitive data store hi nahi karte, data breach ka risk practically zero hai.

  • ✅ Hamaari website HTTPS use karti hai — sabhi connections encrypted hain
  • ✅ No file storage — breach possible hi nahi
  • ✅ No user accounts — credential leak ka risk zero
  • ✅ Browser-only processing — aapka data aapke device par hi rehta hai

6. Aapke Rights (GDPR + Indian IT Act)

Aapko ye rights hain:

  • Right to Access: Aap jaanna chahein ki hum kya data rakhte hain — contact karein
  • Right to Deletion: Newsletter subscribe kiya hai? Hum aapka email delete kar denge
  • Right to Object: Analytics tracking ke against object kar sakte hain
  • Data Portability: Aapka data (agar koi ho) export karne ka request kar sakte hain

In rights ko exercise karne ke liye: privacy@bhaihelp.in

Agar aapko lagta hai koi minor ne data submit kiya hai — humse contact karein aur hum turant delete kar denge.

9. Indian IT Act Compliance

ToolNest Information Technology Act, 2000 aur IT (Amendment) Act, 2008 ke Section 43A ke accordance mein operate karta hai:

  • Hum "Sensitive Personal Data or Information" (SPDI) collect nahi karte
  • Jahan bhi data collect hota hai (newsletter), aapka consent liya jaata hai
  • Data sirf stated purpose ke liye use hota hai
  • Reasonable security practices implement ki gayi hain (HTTPS, no storage)

Governing law: India. Jurisdiction: Courts of India.
