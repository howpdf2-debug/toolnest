How to Use Base64 Encoder/Decoder

Encoding:

  1. Enter text in the left panel or upload a file
  2. Click "Encode" button
  3. Base64 encoded result appears in the same panel
  4. Copy or download the encoded text

Decoding:

  1. Paste Base64 string in the right panel
  2. Click "Decode" button
  3. Original text appears in the same panel
  4. Copy or download the decoded text
