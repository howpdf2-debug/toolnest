How to Create Secure Passwords

  1. Choose Length: Minimum 12 characters recommended (16+ for high security)
  2. Select Character Types: Check boxes for uppercase, lowercase, numbers, symbols
  3. Exclude Ambiguous: Optional - removes characters that look similar (0/O, 1/l/I)
  4. Generate: Click "Generate Password" button
  5. Check Strength: Review strength meter (aim for "Strong")
  6. Copy & Save: Copy to clipboard and save in password manager

Password Strength Guide

  • Weak: 8-11 characters, limited character types
  • Fair: 12-15 characters, 2-3 character types
  • Good: 16-19 characters, 3-4 character types
  • Strong: 20+ characters, all character types
